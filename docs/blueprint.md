# **App Name**: AfriCuisine

## Core Features:

- Immersive Home: Animated background of moving African vector patterns, categories with 3D cards, popular recipes with zoom effect, blog carousel.
- Dynamic Recipes: Animated search bar with pulsating loupe, dynamic filters (country, region, dish type), responsive grid with 3D effect.
- Recipe Detail: Parallax main image with light halo, animated floating ingredient bubbles, interactive timeline steps, 'Miam' and 'Comment' buttons with micro-interactions.
- Create Recipe Form: Form with semi-transparent background, moving vector patterns, dynamic ingredient tags, animated image previews.
- Blog Articles: List of articles with sliding effect and dynamic image, structured content with subtle transitions.
- User Authentication: Glassmorphic design, fluid animations on field validation and focus, luminous avatar.

## Style Guidelines:

- Primary color: Luminous gold (#FFD700) for a vibrant and futuristic feel.
- Background color: Deep black (#0A0A0A) with animated golden African patterns.
- Accent colors: Solar orange (#E57C23), papaya green (#A4DE02), indigo blue (#273C75), terracotta red (#B23A48).
- Titles: 'Playfair' (serif) Bold for an elegant, high-end feel. Note: currently only Google Fonts are supported.
- Text: 'Inter' (sans-serif) for a modern and neutral look. Note: currently only Google Fonts are supported.
- Animated SVGs, Lottie animations, and light Canvas for icons and patterns. Icons with subtle movements and glows.
- Glassmorphism elements for Navbar and forms, 3D RecipeCards with tilt and zoom, floating and rotating elements.
- Use Framer Motion for smooth transitions, micro-interactions, and subtle effects like floating ingredients and page transitions (luminous sweep or desert wind effect).
